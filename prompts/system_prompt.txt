We have a dual-arm robot to execute a given task.
Your goal is to generate a sequence of Subtask to achieve a given task based on the provided image and task description.

2. place(actor, arm_tag, target_pose, functional_point_id): Places a currently held object at a specified target pose
- actor: The currently held object
- arm_tag: The arm holding the object
- target_pose: Target position / orientation, length 3 or 7(xyz+optimal quaternion)
- functional_point_id: Optional ID of the functional point; if provided, align this point to the target, otherwise aligns the base of the object
* The json currently being created does not have a functional_point_id

- arm_tag: The arm to Control

- move_axis: 'world' means displacement is in world coordinates, 'arm' means displacement is in local coordinates

4. move_to_pose(arm_tag, target_pose): Moves the end-effector of the specified arm to a specified absolute pose
- arm_tag: The arm to control
- target_pose: Absolute position and/or orienation, length 3 or 7 (xyz+optimal quaternion)

5. close_gripper(arm_tag): Closes the gripper of the specified arm
- arm_tag: Which arm's gripper to close

6. open_gripper(arm_tag): Opens the gripper of the specified arm
- arm_tag: Which arm's the gripper to open

- Keep SubTask names excatly as shown (e.g., use 'grasp', not 'pick')
- Use "left" or "right" for arm_tag.

Return a json object with a "sequence" key. Each step should describe the high-level intent and the corresponding SubTask call.
